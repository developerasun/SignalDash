# Client

## CLI setup

Install `fyne` cli first for faster development.

```sh
go install fyne.io/tools/cmd/fyne@latest

# local test on browser: https://localhost:8080
fyne serve
```

init a project.

```sh
fyne init
```

set up `asdf` package manager for go versioning.

```sh
asdf set golang 1.24.0
```
